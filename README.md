## Экспорт таблиц из API. Турчанинов Андрей @Cas_Cade

### О программе
Программа получает на вход url API в входной точке проргаммы, после чего она подключается к этому API и берёт данные по методу get

## Предварительные требования
- Python 3.8+

## Установка зависимостей
```bash
pip install -r requirements.txt
```

### Инициализация программы
```bash
python main.py
```

# или с параметром --url
```bash
python main.py --url https://jsonplaceholder.typicode.com/users
```
